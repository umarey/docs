## POST /api/v1/viewer-token

Create a short-lived JWT token that can be used to embed a read-only audit log viewer in your application. Tokens are scoped to specific filters so end users only see relevant events.

### Request Body

<ParamField body="filters" type="object">
  Filter constraints for the viewer session. The token holder can only see events matching these filters.

  <ParamField body="filters.actor_id" type="string">
    Restrict to events by a specific actor.
  </ParamField>

  <ParamField body="filters.resource_id" type="string">
    Restrict to events on a specific resource.
  </ParamField>

  <ParamField body="filters.tenant_id" type="string">
    Restrict to events for a specific tenant.
  </ParamField>

  <ParamField body="filters.action" type="string">
    Restrict to events matching an action pattern.
  </ParamField>
</ParamField>

<ParamField body="ttl" type="integer" default="3600">
  Token lifetime in seconds. Maximum 86400 (24 hours).
</ParamField>

### Response

```json
{
  "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
  "expires_at": "2026-03-26T11:15:00Z"
}
```

### Examples

<CodeGroup>

```bash cURL
curl -X POST https://getimmutable.dev/api/v1/viewer-token \
  -H "Authorization: Bearer imk_your_api_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "filters": {
      "tenant_id": "org_7rT2xBc",
      "resource_id": "doc_8nXpQr3"
    },
    "ttl": 7200
  }'
```

```typescript JavaScript
const response = await fetch("https://getimmutable.dev/api/v1/viewer-token", {
  method: "POST",
  headers: {
    "Authorization": "Bearer imk_your_api_key_here",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    filters: { tenant_id: "org_7rT2xBc" },
    ttl: 7200,
  }),
});

const { token, expires_at } = await response.json();
```

</CodeGroup>

### Usage

Pass the token to your frontend to render the embedded viewer. The viewer fetches events directly from Immutable using the scoped token -- no server-side proxy needed.

```html
<iframe
  src="https://getimmutable.dev/viewer?token=eyJhbGciOiJIUzI1NiIs..."
  width="100%"
  height="600"
  frameborder="0"
></iframe>
```

<Note>
  Viewer tokens are read-only. They cannot be used to ingest events or access any write endpoints.
</Note>
