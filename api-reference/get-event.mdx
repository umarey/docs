## GET /api/v1/events/{id}

Retrieve a single event by its UUID. The response includes the full event data and the `integrity` block containing the event's hash and its link to the previous event.

### Path Parameters

<ParamField path="id" type="string" required>
  The UUID of the event.
</ParamField>

### Integrity Block

Every event includes an `integrity` object:

<ResponseField name="integrity.event_hash" type="string">
  SHA-256 hash of this event's data combined with the previous event hash.
</ResponseField>

<ResponseField name="integrity.previous_event_hash" type="string | null">
  SHA-256 hash of the preceding event in the workspace chain. `null` for the first event.
</ResponseField>

See [Hash Chain](/guides/hash-chain) for details on how hashes are computed.

### Errors

| Status | Description                              |
| ------ | ---------------------------------------- |
| `404`  | Event not found or outside retention window. |
