## Installation

```bash
composer require getimmutable/laravel
```

## Configuration

Publish the configuration file:

```bash
php artisan vendor:publish --tag=immutable-config
```

Add your API key to `.env`:

```env
IMMUTABLE_API_KEY=imk_your_api_key_here
```

## Sending Events

Use the `Immutable` facade with the fluent builder:

```php
use Immutable\Facades\Immutable;

Immutable::event()
    ->actor('user_2hG9kLm', name: 'Sarah Chen', type: 'user')
    ->action('document.created', category: 'documents')
    ->resource('doc_8nXpQr3', name: 'Q4 Financial Report', type: 'document')
    ->metadata(['folder' => 'reports', 'template' => 'quarterly'])
    ->target('folder', 'folder_3mK9pLq', name: 'Shared Reports')
    ->session('sess_4kN8pLm')
    ->tenant('org_7rT2xBc')
    ->send();
```

## Auto-Session Tracking

The Laravel SDK can automatically attach a `session_id` to every event based on the current authenticated session:

```php
// config/immutable.php
return [
    'api_key' => env('IMMUTABLE_API_KEY'),
    'auto_session' => true,
];
```

When `auto_session` is enabled, the SDK reads the session ID from Laravel's session store and includes it on every event.

## Batch Ingestion

```php
use Immutable\Facades\Immutable;

Immutable::sendBatch([
    Immutable::event()
        ->actor('user_2hG9kLm', name: 'Sarah Chen')
        ->action('document.created')
        ->resource('doc_8nXpQr3', name: 'Q4 Financial Report'),
    Immutable::event()
        ->actor('user_2hG9kLm', name: 'Sarah Chen')
        ->action('document.shared')
        ->resource('doc_8nXpQr3', name: 'Q4 Financial Report')
        ->target('team', 'team_5bR7cNx', name: 'Engineering'),
]);
```

## Using with Eloquent Models

Track model events automatically:

```php
use Immutable\Facades\Immutable;

// In an observer or model event listener
class DocumentObserver
{
    public function created(Document $document): void
    {
        Immutable::event()
            ->actor(auth()->id(), name: auth()->user()->name)
            ->action('document.created')
            ->resource($document->id, name: $document->title, type: 'document')
            ->metadata(['size' => $document->size, 'mime_type' => $document->mime_type])
            ->send();
    }
}
```

## Error Handling

```php
use Immutable\Facades\Immutable;
use Immutable\Exceptions\ImmutableException;

try {
    Immutable::event()
        ->actor('user_2hG9kLm')
        ->action('document.created')
        ->send();
} catch (ImmutableException $e) {
    Log::error('Immutable API error', [
        'status' => $e->getCode(),
        'message' => $e->getMessage(),
    ]);
}
```
